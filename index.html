Machine Learning

Credit card fraud detection

Caught 80% of fraud at 87% precision on 285k card transactions (ROC-AUC 0.939). XGBoost + SMOTE on a 0.17% positive class. Numbers below tie directly to executed cells in the GitHub notebook.

80% fraud recall 284,807 rows ROC-AUC ~0.94
Expand

At a glance

284,807 transactions, only 492 fraud (0.17% positive rate), a classic imbalanced problem. Goal: catch fraud (recall) while keeping precision usable. Work is documented in a single runnable Jupyter notebook.

Pipeline

  • Stratified split: train 227,845 / test 56,962
  • RobustScaler on amount and time
  • SMOTE applied to training split to address extreme class imbalance (0.17% positive rate)
  • XGBoost and Random Forest (default settings in the notebook)
  • Reports: confusion matrices, PR/ROC curves

Holdout metrics

  • XGBoost: fraud recall 80%, precision 87%, ROC-AUC 0.939
  • RF: fraud recall 64%, ROC-AUC 0.958
  • Macro recall 90% (sklearn report) averages both classes, not the same as the 80% fraud-class recall

Reading the results

Accuracy is near 100% for both models because negatives dominate, which is why the story is PR-AUC / ROC / recall, not accuracy.

Machine Learning

SpaceX Falcon 9 landing prediction

Binary classifiers on payload and orbit features predicting first-stage landing success. 90% test accuracy via GridSearchCV on a multi-model comparison.

90% accuracy SVM · KNN · LR GridSearchCV
Expand

At a glance

Predicts whether a Falcon 9 first stage will land successfully using launch payload, orbit type, and mission parameters. Goal: compare classification algorithms on a real-world binary outcome with meaningful feature engineering.

Pipeline

  • Feature engineering on payload mass, orbit class, launch site, and booster version
  • Train/test split with stratification on landing outcome
  • Models: Logistic Regression, SVM, KNN
  • GridSearchCV over hyperparameter grids for each classifier
  • Evaluation: accuracy, confusion matrix, classification report

Results

  • Best model: 90% test accuracy via GridSearchCV
  • SVM and LR both competitive; SVM edges out on this dataset
  • Confusion matrix confirms low false-negative rate on landing failures
